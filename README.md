# PlasmIT (marketing site)

Investor- and hospital-facing marketing site for PlasmIT / PiMed, built with Next.js (App Router) and shadcn/ui.

## Tech stack

- Next.js (App Router)
- React
- Tailwind CSS v4
- shadcn/ui + Radix primitives
- Framer Motion (scroll narrative)
- Lenis (smooth scroll)

## Getting started

Install dependencies:

```bash
bun install
```

Run the dev server:

```bash
bun run dev
```

Typecheck + production build:

```bash
bun run typecheck
bun run build
```

## Routes

- `/` homepage narrative
- `/investor-corner` investor thesis
- `/hospital-corner` hospital business case (metrics live here)
- `/pimed` product overview
- `/leadership`, `/vision-and-values`, `/about-us`, `/contact-us`
